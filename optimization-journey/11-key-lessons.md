# 📝 获得的关键经验

1. **上下文效率至关重要**
   - 上下文窗口优化直接影响AI性能
   - 选择性文档加载为复杂处理保留上下文
   - 视觉模式比文本需要显著更少的上下文空间
   - 单一真实来源消除冗余信息加载

2. **视觉处理显著减少认知负荷**
   - 视觉信息处理比文本快约60,000倍
   - 模式识别实现快速流程状态感知
   - 视觉层次结构提高信息密度
   - 标准化视觉标记减少上下文需求

3. **选择性加载胜过综合文档**
   - 仅加载与阶段相关的文档保留上下文空间
   - 及时引用加载防止上下文过度消耗
   - 基于复杂度的文档加载高效扩展
   - 基于引用的系统优于基于重复的系统

4. **流程执行需要最小上下文**
   - 视觉检查点提供高效验证
   - 紧凑流程状态跟踪保留上下文
   - 基于模式的违规检测需要最小开销
   - 视觉警报在没有冗长解释的情况下高效沟通

5. **持续上下文优化推动性能**
   - 定期上下文窗口评估揭示优化机会
   - 上下文监控实现动态调整
   - 最小模式为极端约束提供后备
   - 上下文感知流程扩展适应可用资源
